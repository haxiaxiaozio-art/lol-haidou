# 海斗战报 MVP

网页端海克斯大乱斗赛后复盘原型，支持已登录 LOL 客户端读取、演示数据、CSV/JSON 导入、六职业评分、历史回城死亡规则、白天模式和深夜模式。

## 在线版本

[打开海斗战报在线版](https://lol-haidou.haxiaxiaozio.chatgpt.site)

在线网站可以直接体验演示数据和 CSV/JSON 导入。读取真实 LOL 战绩时，需要在同一台 Windows 电脑上运行 `启动数据助手.cmd`，并在浏览器提示时允许访问本地网络。线上网站只连接明确列入 `helper/allowed-origins.json` 的正式域名。

## 运行

需要 Node.js 22.13 或更高版本。

最简单的方式：先启动 LOL 并登录，再双击项目根目录里的 `启动网站.cmd`。启动脚本会在后台开启仅限本机访问的数据助手；首次运行会自动安装依赖，随后在浏览器打开终端显示的 Local 地址。

也可以在终端运行：

```powershell
npm install
npm run dev
```

如果网页提示找不到数据助手，可单独双击 `启动数据助手.cmd`，保持窗口开启后重新检测。

## 读取 LOL 客户端

1. 启动并登录国服 LOL 客户端，停留在客户端主页即可。
2. 打开海斗战报，选择“本地 LOL 客户端”。
3. 点击“检测登录客户端”，确认 Riot ID 和大区。
4. 选择读取 20、40、100 或 200 场，点击“筛选海斗并生成评分”。

本地数据助手只监听 `127.0.0.1:3211`，只接受本机网页请求，不读取 LOL 或 WeGame 密码，也不会把战绩上传到远端。它读取 League Client 本地接口；该接口不是 Riot 正式支持的稳定公共 API，客户端更新后可能需要同步调整连接器。

正式构建使用：

```powershell
npm run build
```

## 数据导入

打开页面中的“导入自己的 CSV / JSON”，下载模板后替换演示字段。文件只在浏览器页面中读取，不会上传。

CSV 的 `role` 字段仅接受：辅助、法师、刺客、坦克、射手、战士。`augments` 使用竖线分隔。历史回城局可填写 `recall_pick_minute`、`deaths_before_recall` 和 `deaths_after_recall`。
