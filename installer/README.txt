HaiDou Data Helper 0.5.0

This helper runs only on 127.0.0.1:3212 and reads the signed-in League Client on this Windows PC.
It does not request or store a League or WeGame password.
The installer asks you to choose a writable folder and installs only for the current Windows user.
It does not request administrator permission or write to Program Files.

After installation:
1. Start and sign in to League of Legends.
2. Open https://haxiaxiaozio-art.github.io/lol-haidou/?v=7
3. Use My Match History or Player Search.

Uninstall from Windows Settings > Apps > Installed apps > HaiDou Data Helper.
